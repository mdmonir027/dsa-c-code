# Algorithm to compare three number;

```
1. Start
2. Input num1 ,num2 ,num3
3. if num1 > num2 and num1 > num3, big = num1
4. if num2 > num1 and num2 > num3, big = num2
5. if num3 > num2 and num3 > num1, big = num3
4. Output big
5. End
```
